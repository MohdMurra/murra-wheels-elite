# Murra Wheels Elite
A modern luxury car showroom and dealership demo built with React, TypeScript, Vite, Tailwind CSS, and shadcn-ui.

## Project topic
Luxury automotive showroom and dealership experience.

## Project idea
Murra Wheels Elite is designed as a polished front-end prototype for a premium car dealer. It showcases a curated inventory of luxury vehicles, provides detailed vehicle pages, and supports user interactions for browsing inventory, scheduling test drives, and reviewing trade-in options. The project also includes an admin dashboard for managing car listings, users, messages, and bookings.

## Overview
This project delivers a responsive, visually appealing user interface for a dealership experience, including:
- Inventory browsing with filters for brand, body style, and price
- Featured vehicle highlights and vehicle detail pages
- User-facing pages for sign in, registration, contact, and service information
- Test drive and trade-in request flows
- Admin dashboard navigation with overview, car listings, user management, messages, and bookings

## Tech stack
- React 18
- TypeScript
- Vite
- Tailwind CSS
- shadcn-ui + Radix UI primitives
- React Router DOM
- @tanstack/react-query
- Lucide icons
- Recharts
- React Hook Form + Zod

## Features
- Luxury car inventory with filters
- Featured vehicle showcase
- Vehicle detail pages with specs and images
- User-facing forms for contact, test drives, and trade-in requests
- Admin dashboard with overview, listings, users, messages, and bookings
- Responsive design and modern UI interactions

## Notes
- Inventory data is currently static in `src/data/cars.ts`.
- The current experience is primarily front-end focused and ready for backend integration.
- Customize the dataset, update images, or connect APIs for a production-ready dealership application.
